## üõ°Ô∏è What is VALHALLA?

**VALHALLA** is not just another to-do list. It is a **gamified life-management ecosystem** designed for those who treat discipline as a battle and productivity as a conquest.

Built with the latest web technologies (**React, Vite**) and wrapped for native performance (**Electron, Capacitor**), VALHALLA runs everywhere‚Äîoffline, secure, and fast. It combines a Task Manager, Habit Tracker, Gym Log, and Note-taking system into one cohesive, epic interface.

## ‚öíÔ∏è The Forge (Tech Stack)

Built with a **Local-First** architecture. Your data belongs to you.

| Layer | Technology | Description |
| :--- | :--- | :--- |
| **Core** | React 18 + TypeScript | Blazing fast UI with type safety. |
| **Styling** | Tailwind CSS | Modern, responsive, dark-mode native design. |
| **Build Tool** | Vite | Instant server start and optimized bundling. |
| **Desktop** | Electron | Native Windows (.exe) wrapper. |
| **Mobile** | Capacitor | Native Android (.apk) wrapper. |
| **Database** | LocalStorage / IndexedDB | Offline-ready persistence. No server required. |
| **Charts** | Recharts | Beautiful data visualization. |

## ‚ö° Summoning (Installation)

### Prerequisites
*   Node.js (v16+)
*   Git

### 1. Clone the Repository
```bash
git clone https://github.com/your-username/valhalla.git
cd valhalla
```

### 2. Install Dependencies
```bash
npm install
```

### 3. Run in Development Mode
```bash
npm run dev
```
